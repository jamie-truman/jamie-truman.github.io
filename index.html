What is the difference between Margin, Border, and Padding?

Margin, Border and padding are all functions to control how much space there is in and around the content. These functions are very important in CSS because they give you the ability to make you site look clean and uncluttered.

This image is a very useful representation of how each function works

Padding

Padding is used to make space around content inside of a border. You can adjust any side of the content using each padding function individually padding-top:, padding-right:, padding-bottom:,padding-left: or padding: 0px 0px 0px 0px.

Border

Border is the function that lets you create a border around the content. To do this you use all of the border functions some of which are border:, border-style:, border-color: etc.

Margin

Margin is the function that lets you create space around the outside of the contents borders, This is usully used to seperate differnet pieces of conent on the page. Some of the margin functions are margin-top:, margin-right:, margin-bottom:, margin-left: or margin: 0px 0px 0px 0px.
